# Terminal Resume

A retro CRT terminal-styled personal resume site. Green-on-black, scanlines, typewriter animations, and numbered navigation — like a text adventure for your career.

## Tech Stack

Vite + React 19 + TypeScript + VT323 font + pure CSS

## Getting Started

```bash
npm install
npm run dev
```

## Edit Content

All resume content lives in `src/data/resume.ts`. Update that file — no component changes needed.

## Build & Deploy

```bash
npm run build
```

Output goes to `dist/`. Deploy as a static site to Vercel, Netlify, or GitHub Pages.
